Declaring Variables

In Python, declaring a variable is straightforward. You simply choose a variable name and assign a value to it using the equals sign =.
There's no need to explicitly declare the type of the variable; Python determines the type based on the assigned value.

The example is given as: 

#Integer variable
age = 25
                  
#Float variable
height = 5.9
                  
#String variable
name = "Alice"

#Boolean variable
value=True

Naming rules:

Must start with a letter or underscore, followed by letters, digits, or underscores.
Varibles in python are case-sensitive hense all the variables should be declared carefully.
Variables should be named in such a way that any other programmer that reads our program understands what value in that particular variable is stored.

Characteristics:

  • Mutable and Immutable
    • Mutable:
      • Values that can be changed after assignment (e.g., lists, dictionaries).
    • Immutable
      • Values that cannot be changed after assignment (e.g., integers, strings, tuples)
  • No Explicit Scope Declaration:
    • The scope of a variable depends on where it is defined:
    • Global:
      • Declared outside of functions, accessible everywhere.
    • Local:
      • Declared within a function, accessible only within that function.

Conclusion

Understanding how to declare variables properly is crucial for writing clean, maintainable, and error-free Python code. Choosing the right words based on the value of variables helps to maintain code and makes code more readable.
