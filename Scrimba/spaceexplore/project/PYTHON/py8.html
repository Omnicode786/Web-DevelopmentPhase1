Strings in Python

Strings are a fundamental data type in Python, used to represent text. A string is a sequence of characters enclosed in single quotes ('...'), double quotes ("...") or triple quotes ("""...""") for multi-line strings or docstrings.

1. Defining a String

Strings can be created using different types of quotes:

singleQuoteString = 'Hello, World!'
doubleQuoteString = "Hello, World!"
templateLiteral = """Hello, World!"""
  • Single and Double Quotes: Both can be used interchangeably. The choice is mostly a matter of preference or consistency.
  • Docstrings: These are enclosed in triple quotes and allow for multi-line strings and string interpolation.

2. String Properties

  • Len: The "len" property returns the number of characters in a string.

Example:

Str = "Hello"
print(len(Str)) # Output: 5

3. String Methods

Pyhton provides numerous methods to manipulate and interact with strings. Here are some of the most commonly used string methods:

3.1 lower()

The "variable.lower()" method returns the converts the string to lowercase

Str = "Hello"
print(Str.lower()) # Output: "hello"

3.2 upper()

The "upper()" method converts the string to uppercase.

Str1 = "Hello"
result = Str1.upper()
print(result) # Output: "HELLO"

3.3 strip()

The "strip()" method removes leading and trailing whitespace from the string.

Str = "  Hello, World!  "
print(Str.strip()) # Output: "Hello, World!"

3.4 split()

The "split()" method splits the string into a list of substrings based on a delimiter. By default it breaks on spaces.

str = "Hello, World!";
print(Str.split()) # Output: ["Hello,", "World!"]

3.5 replace()

The "replace(old, new)" method replaces occurrences of a substring with another substring. It takes 2 arguments, 1st the word you want to replace and 2nd the word with which you want to replace.

Str = "Hello, World!"
result = Str.replace("World", "Universe")
print(result) # Output: "Hello, Universe!"

3.6 find()

The "find(substring)" method returns the index of the first occurrence of a substring, or -1 if not found.

Str = "Hello, World!"
result = Str.find("World")
print(result) # Output: 6

3.7 startswith()

The "startswith(prefix)" method checks if the string starts with a specified prefix.

Str = "Hello, World!"
print(Str.startswith("Hello")) # Output: True

3.8 endswith()

The "endswith(suffix)" method checks if the string ends with a specified suffix.

Str = "Hello, World!"
print(Str.(World!)) # Output: True

3.9 isdigit()

The "replace()" method checks if all characters in the string are digits.

Str = "1234"
result = Str.isdigit()
print(result) # Output: True

3.10 isalpha()

The "isalpha()" method checks if all the characters in a string are alphabetic (i.e., letters only) and that the string is not empty. If the string contains only alphabetic characters, isalpha() returns True; otherwise, it returns False.

Str = "Hello, World!"
result = Str.isaplha()
print(result) # output: False
Str2="HelloWorld"
result=Str2.isaplha()
print(result) # output: True

4. Docstrings

Docstrings are special multi-line strings used to document Python modules, classes, and functions. They are enclosed in triple quotes and placed as the first statement within these constructs. Docstrings provide descriptions that can be accessed through the help() function or the __doc__ attribute.

Example:

multiLineString = '''This is
a multi-line string'''
print(multiLineString)

Output:

This is
a multi-line string.

Conclusion

Strings in Python are versatile and come with a wide range of methods that make text manipulation easy and powerful. Whether you're working with simple text or complex data, understanding how to use these string methods will help you write more efficient and readable code.
