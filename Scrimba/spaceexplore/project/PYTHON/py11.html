Dictionaries in Python

Dictionaries in Python are unordered collections that store data in key-value pairs. Each key in a dictionary is unique, and it maps to a specific value. Dictionaries are mutable, meaning you can change, add, or remove key-value pairs after the dictionary has been created.

Key Characterstics:

  • Key-Value Pairs: Each item in a dictionary is a key-value pair. The key acts as an identifier, and the value is the data associated with that key.
  • Mutable: You can modify a dictionary by adding, updating, or removing items.
  • Ondered: Unlike lists, dictionaries do not maintain any order for their items.
  • Keys Must Be Immutable: Keys can be of any immutable data type, such as strings, numbers, or tuples.

1. Creating a dictionary

You can create a dictionary using curly braces '{}' with key-value pairs separated by a colon:

my_dict = {"name": "Alice", "age": 30, "city": "New York"}
  • "name", "age", "city" are keys
  • "Alice", 30, "New York" are pairs

2. Accessing Values

You can access values by referring to their corresponding keys:

Example:

print(my_dict["name"])  # Output: Alice

3. Modifying Dictionaries

You can update a value associated with a key:

Example:

my_dict["age"] = 31

You can also add new key-value pairs:

Example:

my_dict["email"] = "alice@example.com"

4. Removing Elements

To remove a key-value pair, you can use the 'del' keyword or the 'pop()' method:

del my_dict["city"]
email = my_dict.pop("email")
print(my_dict) # output: {"name": "Alice", "age": 31}

5. Looping Through a Dictionary

You can iterate through a dictionary's keys, values, or key-value pairs:

for key in my_dict:
    print(key, my_dict[key])
        # or
for key, value in my_dict.items():
    print(key, value)

Conclusion

Dictionaries in Python store data as key-value pairs, allowing efficient access and modification. They are mutable, meaning you can change their contents, and keys must be unique and immutable. Dictionaries are essential for mapping unique identifiers to specific values, making them highly versatile in programming.
