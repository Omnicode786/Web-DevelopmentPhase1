Python Data Types

Python is a dynamically-typed language, which means you don't need to specify the data type of a variable when you declare it. However, understanding the different data types in Python is crucial for writing effective and bug-free code. Python has several built-in data types, which can be categorized into primitive and non-primitive types.

Primitive Data Types

Primitive data types are the most basic types of data. They are immutable, meaning their value cannot be changed once created.

Characteristics:

  1. Int and Float:
    • Represents both integer and floating-point numbers. 
      age = 25 #"int"
temperature = -10.5 #"float"
  2. String:
    • Represents a sequence of characters used to store text.
    • Strings can be enclosed in single quotes or double quotes. 
      greeting = "Hello, world!"
name = 'Alice'
message = f"Welcome, {name}!"
  3. Boolean:
    • Represents a logical entity and can have two values: "True" and "False". 
      isActive = True
isCompleted = False

    4. Non-Primitive Data Types

    Non-primitive data types are objects, and they can hold collections of values or more complex entities.

    • Dictionary:
      • Represents a collection of key-value pairs.
      • Objects can hold other objects, primitive values, functions, and more. 
        person = {
"name": "Alice",
"age": 30,
"isStudent": True
}
    • List:
      • Represents an ordered list of values (elements).
      • Lists can hold any data type, including other Lists or dictionaries. 
        numbers = [1, 2, 3, 4, 5]
MixedList = [1, "Hello", true]
    • Function:
      • Represents a block of code designed to perform a particular task.
      • Functions are objects in Python and can be assigned to variables, passed as arguments, and returned from other functions. 
        def add(a, b):
    return a + b

    Type Checking

    In Python, you can check the data type of a variable or value using the built-in type() function. This function returns the type of the object passed to it.

    print(type(42)) # "int"
print(type(42.0)) # "float"
print(type("Hello")) # "string"
print(type(True)) # "boolean"
print(type({})) # "Dict"
print(type([])) # "List" (Lists are a type of object)

    Conclusion

    Understanding Python data types is fundamental for writing effective code. By knowing the different types and how to use them, you can avoid common pitfalls and ensure your programs handle data correctly.
