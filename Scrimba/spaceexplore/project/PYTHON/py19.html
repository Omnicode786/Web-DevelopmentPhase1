File handling

File handling in Python involves reading from and writing to files on your computer. Python provides built-in functions and methods that allow you to manipulate files easily.

Opening a file:

In Python, the 'open()' function is used to open a file.

Syntax:

file_object = open(filename, mode)

Explanation:

  • filename:The name (and path) of the file to open.
  • mode: A string that specifies the mode in which the file is opened. Common modes include:
    • 'r':Read (default mode) - opens the file for reading.
    • 'w':Write - opens the file for writing (creates a new file or truncates an existing file).
    • 'a':Append - opens the file for writing, appending to the end of the file if it exists.
    • 'b':Binary mode - opens the file in binary mode (e.g., 'rb' or 'wb').
    • '+':Update mode - allows both reading and writing (e.g., 'r+', 'w+').

Reading from a file:

You can read the contents of a file using several methods:

  • read(): Reads the entire file

    • Example:

    with open('example.txt', 'r') as file:
content = file.read()
print(content)
  • readline(): Reads one line at a time

    • Example:

    with open('example.txt', 'r') as file:
line = file.readline()
while line:
print(line, end='')
line = file.readline()
  • readlines(): Reads all lines into a list.

    • Example:

    with open('example.txt', 'r') as file:
lines = file.readlines()
for line in lines:
print(line, end='')

Writing to a File

To write to a file, you use the 'write()' method or the 'writelines()' method:

  • write() Writes a string to the file.

    • Example:

    with open('example.txt', 'w') as file:
file.write('This is a line of text.\n')
  • writelines() Writes a list of strings to the file.

    • Example:

    with open('example.txt', 'w') as file:
lines = ['First line.\n', 'Second line.\n', 'Third line.\n']
file.writelines(lines)

Appending to a file

If you want to add content to the end of a file without overwriting it, you can open the file in append mode ('a'):

Example:

with open('example.txt', 'a') as file:
file.write('This will be added to the end of the file.\n')

Closing a file:

While Python automatically closes the file when using the with statement, you can also manually close a file using the close() method:

file = open('example.txt', 'r')
content = file.read()
file.close()

Working with binary files:

To read or write binary files (such as images or executables), use the 'b' mode:

Example:

with open('image.png', 'rb') as file:
data = file.read()
with open('new_image.png', 'wb') as file:
file.write(data)

Conclusion

File handling in Python allows for efficient reading, writing, and appending to files using the open() function with different modes. The with statement ensures files are safely closed after operations, and exception handling prevents errors. This makes Python's file handling both simple and robust for managing data.
