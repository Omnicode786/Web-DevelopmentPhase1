Sets in Python

A set is an unordered collection of unique elements in Python. Unlike lists or tuples, sets do not allow duplicate values, and their elements are not stored in any particular order. Sets are mutable, meaning you can add or remove items after creation, but the elements themselves must be immutable (like strings, numbers, or tuples).

Key Characterstics:

  • Unordered: Sets do not maintain the order of elements. This means that the items have no index, so you cannot access elements by position.
  • Unique: Sets automatically remove duplicate values.
  • Immutable Elements: The elements of a set must be immutable (e.g., numbers, strings, or tuples), but the set itself is mutable, allowing you to add or remove items.

Syntax:

You can create a set using curly braces '{}' or the 'set()' function.

# Creating a set with curly braces
my_set = {1, 2, 3, 4, 5}
            
# Creating a set with the set() function
another_set = set([1, 2, 3, 4, 5])

Common set operations:

  • Adding elements: Use 'add()' to add a single element to a set.
    • my_set = {1, 2, 3}
my_set.add(4)  # Output: {1, 2, 3, 4}
  • update(): Use 'update()' to add multiple elements
    • my_set.update([5, 6])
# output: {1, 2, 3, 4, 5, 6}

Removing Elements:

Removing elements from a set can be done by few ways.

  • remove(): Use remove() to remove a specific element (raises an error if the element does not exist).
  • discard(): Use discard() to remove an element without raising an error if the element is not found.
  • pop(): Use pop() to remove and return an arbitrary element.
  • clear(): Use clear() to remove all elements from the set.
  • pop(): Use pop() to remove and return an arbitrary element.

Example:

my_set.remove(2)  # Removes 2 from the set
my_set.discard(3)  # Removes 3 from the set
my_set.pop()  # Removes and returns an arbitrary element
my_set.clear()  # Removes all elements from the set

Set operations:

  • Union: Combines elements from two sets, returning a set with all unique elements from both.
    • set1 = {1, 2, 3}
set2 = {3, 4, 5}            
union_set = set1.union(set2)  # output: {1, 2, 3, 4, 5}
  • intersection: Returns a set containing only the elements found in both sets.
    • set1 = {1, 2, 3}
set2 = {3, 4, 5}            
intersection_set = set1.intersection(set2)  #  output: {3}
  • difference: Returns a set with elements that are in the first set but not in the second.
    • set1 = {1, 2, 3}
set2 = {3, 4, 5}            
difference_set = set1.difference(set2)  # output {1, 2}
  • Symmetric difference: Returns a set with elements that are in either of the sets but not in both.
    • set1 = {1, 2, 3}
set2 = {3, 4, 5}            
sym_diff_set = set1.symmetric_difference(set2)  # output {1, 2, 4, 5}

Conclusion

Sets are a powerful tool in Python when working with unique elements, performing mathematical operations, or efficiently checking membership. They are versatile and can be combined with other data structures for more complex tasks.
