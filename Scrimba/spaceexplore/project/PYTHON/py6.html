Loops in Python

Loops are fundamental structures in programming that allow you to execute a block of code repeatedly, either for a specific number of times or until a certain condition is met. Python provides two types of loops, each suited for different tasks.

1. For loop

The "for" loop is used when you know the number of iterations in advance. It's the most commonly used type of loops in Python.

Syntax:

for i in range(starting num, last num + 1):
    #Code to be executed

Example:

for i in range(0, 5):
    print("Iteration " + i)

Explanation:

  • Initialization: "for i in range(0,5)" initializes i to 0
  • Condition: checks if i<5
  • Increment: increases the counter 1 time after each iteration.

Output:

Iteration 0
Iteration 1
Iteration 2
Iteration 3
Iteration 4

2. While loop

The "while" loop is used when you want to repeat a block of code as long as a condition is true. It's useful when the number of iterations isn't known beforehand.

Syntax:

while (condition):
    #Code to be executed

Example:

i = 0
while (i < 5):
    print("Iteration " + i)
    i+=1

Explanation:

  • The loop continues to execute as long as "i < 5" is true.
  • Don't forget to increment the counter (i+=1) to avoid an infinite loop.

Nested loops

You can nest loops within each other, creating a loop inside another loop. This is useful for working with multi-dimensional lists or performing complex iterations.

Example:

for i in range(1, 4):
    for j in range(1, 4):
        print(f"i={i}, j={j}")

Output:

i = 1, j = 1
i = 1, j = 2
i = 1, j = 3
i = 2, j = 1
i = 2, j = 2
i = 2, j = 3
i = 3, j = 1
i = 3, j = 2
i = 3, j = 3

Conclusion

Python offers 2 tyoes of loop structures that are versatile and powerful, allowing you to handle different types of iterative tasks efficiently. Whether you're working with lists, dictionaries, or need to repeat code until a condition is met, there's a loop for your needs. Understanding when and how to use each loop will make your code more efficient and easier to maintain.
