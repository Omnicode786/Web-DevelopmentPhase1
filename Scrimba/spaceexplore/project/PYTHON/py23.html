Patterns Regarding Loops

Unleash your creativity with Python! This concise Python script generates intricate different patterns effortlessly. By adjusting a single parameter, you can craft patterns of varying sizes, making it perfect for decorative elements, educational exercises, or purely artistic pursuits. The script employs nested loops and string manipulation techniques to ensure each pattern maintains its symmetrical allure, resulting in visually captivating designs. Whether you're a beginner or an experienced coder, this Python code opens up a world of geometric possibilities, inviting you to explore and create mesmerizing patterns with ease.

Diamond Pattern:

A Neat and Symmetrical Output!

lines = int(input("Enter the number of lines: "))
                        
for i in range(1, lines + 1):
      print(" " * (lines - i) + "*" * (2 * i - 1))
for j in range(lines - 1, 0, -1):
      print(" " * (lines - j) + "*" * (2 * j - 1))

Output:

  *  
 ***
*****
 ***
  *

Pyramid Pattern:

A Perfectly Balanced Design!

n = int(input("Enter the number of rows: "))
for i in range(n):
      print(' ' * (n - i - 1) + '*' * (2 * i + 1))

Output:

     *
    ***
   *****
  *******
 *********

Inverted Pyramid:

A Symmetrical and Compact Pattern!

n = int(input("Enter the number of rows: "))
