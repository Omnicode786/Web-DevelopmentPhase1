"map()", "filter()", "reduce()", and "enumerate()" Functions in Python

1. enumerate() Function

The enumerate() function in Python is a built-in function that adds a counter to an iterable and returns it as an 'enumerate' object. This object can then be used in a for loop to iterate over both the index and the value of each element in the iterable.

Syntax:

enumerate(iterable, start=0)
  • iterable: The 'iterable' you want to iterate over (e.g., a list, tuple, string).
  • start (optional): The index to start counting from. The default is '0'.

Example:

fruits = ["Apple", "Banana", "Cherry"]
for index, value in enumerate(fruits):
    print(f"Index: {index}, Value: {value}")
# Output:
#Index: 0, Value: apple
#Index: 1, Value: banana
#Index: 2, Value: cherry

In this example, "enumerate" iterates over the "fruits" list and logs each element along with its "index" to the console.

2. map() Function

The map() function applies a given function to all items in an iterable and returns a map object, which can be converted to a list

Example:

numbers = [1, 2, 3, 4, 5]
squared_list = list(map(lambda x: x ** 2, numbers))
print(squared_list) # Output: [1, 4, 9, 16, 25]

3. filter() Function

The "filter()" function creates a new list with all elements that pass the test implemented by the provided function. It does not modify the original list.

Example:

def is_even(n):
    return n % 2 == 0     
my_list = [1, 2, 3, 4, 5, 6]
filtered_list = list(filter(is_even, my_list))
print(filtered_list)
# Output: [2, 4, 6, 8, 10]

4. reduce() Function

These methods can be used together to perform complex operations on lists.

Example:

from functools import reduce

numbers = [1, 2, 3, 4, 5, 6]

# 1. Using map to square each number
squared_numbers = list(map(lambda x: x ** 2, numbers))

# 2. Using filter to keep only even squares
even_squared_numbers = list(filter(lambda x: x % 2 == 0, squared_numbers))

# 3. Using enumerate to print index and value
for index, value in enumerate(even_squared_numbers):
    print(f"Index: {index}, Value: {value}")

# 4. Using reduce to sum all even squares
sum_of_even_squares = reduce(lambda x, y: x + y, even_squared_numbers)
print(f"Sum of even squares: {sum_of_even_squares}") # output: 220
