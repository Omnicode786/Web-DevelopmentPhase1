Spread Operator (...)

The Spread Operator allows you to expand elements of an iterable (like arrays or objects) where multiple elements are expected. It's useful for copying, combining, or passing array elements as individual arguments.

1. Expanding Arrays

The spread operator is commonly used to copy arrays or combine multiple arrays into one.

Copying an Array:

let originalArray = [1, 2, 3];
let copiedArray = [...originalArray];
console.log(copiedArray); // [1, 2, 3]

Merging Arrays:

let array1 = [1, 2];
let array2 = [3, 4];
let combinedArray = [...array1, ...array2];
console.log(combinedArray); // [1, 2, 3, 4]

2. Spreading in Function Calls

The spread operator can be used to pass array elements as individual arguments to a function.

Example:

function sum(a, b, c) {
    return a + b + c;
}

let numbers = [1, 2, 3];
console.log(sum(...numbers)); // 6

3. Spreading in Objects

The spread operator also works with objects, making it useful for copying or merging objects.

Copying an Object:

let originalObject = { name: 'Alice', age: 25 };
let copiedObject = { ...originalObject };
console.log(copiedObject); // { name: 'Alice', age: 25 }

Merging Objects:

let obj1 = { name: 'Alice' };
let obj2 = { age: 25 };
let combinedObject = { ...obj1, ...obj2 };
console.log(combinedObject); // { name: 'Alice', age: 25 }

Rest Parameters (...)

The Rest Parameters syntax also uses ..., but it functions differently. It allows a function to accept an indefinite number of arguments as an array.

1. Using Rest Parameters in Functions

Rest parameters collect all remaining arguments passed to a function into a single array. They must be the last parameter in a functionâ€™s argument list.

Example:

function multiply(multiplier, ...numbers) {
    return numbers.map(num => num * multiplier);
}

console.log(multiply(2, 1, 2, 3)); // [2, 4, 6]

In this example, "...numbers" gathers all arguments after multiplier into an array.

2. Difference Between Spread Operator and Rest Parameters

The spread operator expands elements of an iterable, while rest parameters gather arguments into an array.

Spread Example:

let arr = [1, 2, 3];
console.log([...arr, 4, 5]); // [1, 2, 3, 4, 5]

Rest Example:

function add(...args) {
    return args.reduce((acc, curr) => acc + curr, 0);
}
console.log(add(1, 2, 3, 4)); // 10

3. Combining Spread Operator and Rest Parameters

You can combine both spread and rest in your code to make functions more flexible and concise.

Example:

function sortNumbers(...numbers) {
    return [...numbers].sort((a, b) => a - b);
}

console.log(sortNumbers(5, 3, 8, 1)); // [1, 3, 5, 8]

Conclusion

The Spread Operator and Rest Parameters provide powerful ways to work with arrays, objects, and functions. The spread operator helps expand elements, while rest parameters gather them, allowing for clean and concise code.
